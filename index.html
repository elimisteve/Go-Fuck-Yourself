brainfuck.go is a cool, social way to hurt yourself? http://github.com/gfranxman/Go-Fuck-Yourself

Install

$ # compile with go compiler
$ 6g brainfuck.go

$ # then link it
$ 6l -o brainfuck brainfuck.6

$ # then play
$ ./brainfuck tests/99bottles.bf
